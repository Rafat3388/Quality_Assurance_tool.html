Tab 1 — Setup (Supervisors only)
•	Download a pre-filled CSV template for analyte parameters (CD, S_WRL, S_r, No. of Levels, Power Goal)
•	Import or manually enter analytes
•	Automatically computes CD/S_WRL and S_r/S_WRL ratios
•	Selects Table A1, A2, or A3 based on the number of concentration levels (1, 2, or 3)
•	Returns: Rejection Limit (k × CD), Number of Samples (N), and achieved Power
•	Downloadable checklist CSV and print-ready output
Tab 2 — Evaluation (All users)
•	Load protocol from Tab 1 or import the checklist CSV
•	Enter paired Current Lot / Candidate Lot results per sample (live difference calculation)
•	Calculates Mean Difference per level, compares against the RL
•	Pass/Fail per analyte level + Overall Accept/Reject summary
•	Downloadable evaluation template and results report
